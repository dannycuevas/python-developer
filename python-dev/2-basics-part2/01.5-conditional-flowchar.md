# 1️⃣ Flowchart-style diagram explanation

🔁 Conditional Logic — Flowchart Mental Model

Think of Python conditionals as a **decision flow**.

---

### 🧠 How Python Thinks (Step by Step)

```text
        START
          |
          v
   ┌───────────────┐
   │ IF condition? │
   └───────────────┘
          |
   ┌──────┴──────┐
 True           False
  |               |
  v               v
Run IF block   ┌───────────────┐
               │ ELIF condition?│
               └───────────────┘
                       |
                ┌──────┴──────┐
              True           False
               |               |
               v               v
        Run ELIF block      ┌───────────┐
                            │ ELSE block│
                            └───────────┘
                                   |
                                   v
                                 END
```

## 🔑 Key Flow Rules

- Python checks conditions **top → bottom**    
- The **first True** condition wins    
- Once a block runs, Python **exits** the conditional    
- `else` runs **only if everything else is False**    

## 🧠 Memory Trick

> Python does **not** ask _“Which is best?”_  
> It asks _“Which is True first?”_
