# What is the Internet?
- It is a Global Network, of networks (just a bunch of connected computers)

- Computers connected together
- Routing = routers connected computers together
- IP Addressing to Identify Machines
- Network of Networks, and/or small networks connected to other networks, etc
- And all data is sent over the internet
- Think of it like massive network of highways or freeways around the entire globe, connecting all the machines around the world

#### -**Servers** need to be operational 24/7
What they do is, they serve you all the data and files you need to be able to access and interact with certain websites

#### -**Clients** are whatever type of computer that a User would use to access the Internet 
 This is just another term referring to the machine, the computer, the device that is making the Request for something from a Server
	- Like our laptop visiting a website
