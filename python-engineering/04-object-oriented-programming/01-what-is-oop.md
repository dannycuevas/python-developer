# Object Oriented Programming (OOP)

#### **Everything in Python is an object**

- We have that `class` keyword in front of all our Data Types
- So, "everything is an object" because in Python, everything is built by this `class` keyword

```python
print(type(None))
print(type(True))
print(type(5))
print(type(5.5))
print(type('hi'))
print(type([]))
print(type(()))
print(type({}))

# the output:
<class 'NoneType'>
<class 'bool'>
<class 'int'>
<class 'float'>
<class 'str'>
<class 'list'>
<class 'tuple'>
<class 'dict'>
```

- And we are able to use different "methods" in our "objects", for example `some_string.capitalize()` to be able to perform some actions on them

#### **Thus, what is an object?**
- "Objects have methods" like `string.capitalize()` and attributes that you can access using "methods" too

- Meaning, "object oriented programming" allows us to go beyond what Python just gives us, which are the Data Types
- Also, Python will allow us to "create our own Data Types", with different attributes and methods
